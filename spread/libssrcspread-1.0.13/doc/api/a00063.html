Detailed Description

The ssrc namespace is an alias intended for use by library users.

The alternate namespaces prefixed by ssrc::spread_vX_X_X contain versioning information internal to the library and should not be used outside of it. For example, you should use the namespace ssrc::spread instead of ssrc::spread_vX_X_X.
